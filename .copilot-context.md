## Project Overview

**Project:** Financial Manager — US Tax Calculator
**Purpose:** A modern web-based US federal income tax calculator with a Python/FastAPI backend and React/TypeScript frontend.
**Status:** Active — Initial Development

---

## Architecture

### Backend (Python + FastAPI)
- `src/financial_manager/engine/` — Tax calculation logic (brackets, deductions, credits)
- `src/financial_manager/models/` — Pydantic models for tax input/output
- `src/financial_manager/data/` — Tax bracket data, constants, and rate tables
- `src/financial_manager/api/` — FastAPI REST endpoints

### Frontend (React + Vite + TypeScript)
- `frontend/src/components/` — React UI components
- `frontend/src/services/` — API client layer
- `frontend/src/hooks/` — Custom React hooks
- `frontend/src/types/` — TypeScript type definitions

---

## Key Concepts

### Filing Statuses
- Single
- Married Filing Jointly (MFJ)
- Married Filing Separately (MFS)
- Head of Household (HoH)
- Qualifying Surviving Spouse (QSS)

### Tax Calculation Flow
1. User enters gross income, filing status, deductions, credits
2. Backend computes AGI → Taxable Income → Tax Liability → Credits → Amount Owed/Refund
3. Frontend displays results with bracket breakdown
